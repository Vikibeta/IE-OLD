#IE-FBI-WARNing

_____

在开发网站的过程中，不管是优雅降级和渐进增强，总有1款IE 不适合你，然而为了更好的用户体验，你肯定需要一款面对我不兼容浏览器脚本的提示，已提交网站的健硕和友好的交互，由此你应该用的到
IE-FBI-WARNing

这是通用的一个在低版本浏览器上以全屏形式友好提示用户
更换浏览器的小脚本，展示效果如下：
